Edit a new command

To edit command, walk a python file that can be found in '../Script'

For example, we go edit the 'command3.py' file

code

# Command3.py\n from Bash.command import Command\n command = Command()\n\n #create new command\n @command.add(only_one_value=str)\n def mkdir(val):\n \tprint('create file:', val)

#1 Import the Command class

Use the keword 'import' and 'from' for import the Command class

from Bash.command import Command

#2 Init new Command instance

Create variable who will keep the 'Command' class

command = Command()

#3 Create a decorator 'command.add()'

Create decorator for say to the package it's a command

@command.add()

The different param are...

# one_only_value => just a value after keyword: mkdir project1\n\n @command.add(one_only_value=str)\n def mkdir(value):\n \tpass
# one_only_param => just a param after keyword: node -v\n\n @command.add(one_only_param=str)\n def node():\n \tprint("V19.0.1")
# **value => all param + all value: system -ip 127.0.0.1\n\n @command.add(ip=str)\n def system(value):\n \tprint("the system ip changed to", value['ip'])\n\n # you can put as many parameters as you want\n\n @command.add(ip=str, version=str, name=str, etc...)\n def boot(value):\n \tif value['ip'] and value['version'] and value['name']:\n \t\treturn 200\n \telse:\n \t\treturn 404
